# Odoo Play

Welcome to Odoo Play, the visual guide that introduces the fundamental concepts of [Odoo](https://www.odoo.com/) in a simple way. The goal is to illustrate step by step how Odoo allows you to manage the typical business situations which occur every day. The guide makes extensive use of screenshots: they are made at high resolution so as to be easily readable and therefore useful for the purpose.
